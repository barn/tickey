Tickey
===

Tickey is a tool to submit issues to redmine.

Getting Started
---
* `gem install trollop`
* `gem install redmine_client`

Usage
---

    Options:
      --subject, -s <s>:   Subject line for ticket
            --debug, -d:   Be noisey
         --apitoken, -a:   API token to use for redmine
              --url, -u:   URL to Redmine/Chili
             --help, -h:   Show this message
